7 gt Example 1

Here is an example on how to add a title and subtitle using gt

library(tidyverse)
library(gt)

head(mtcars) %>%
  gt()  %>%
  tab_header(
    title = md("Just a **title**"),
    subtitle = md("Just a `subtitle`")
  )
